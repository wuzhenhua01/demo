# 项目说明
快速标记当前位置的安卓地图应用

## 目录规则
根目录禁止放任何java、scala文件，只能放gradle相关文件。

## 技术栈
- 地图SDK：高德地图
- 其他：Kotlin、Jetpack Compose

## 运行环境信息
- python                 - E:\python-3.8.9
- nodejs                 - E:\node-v22.22.1-win-x64
- andorid platform-tools - E:\platform-tools
- andorid cmdline-tools  - E:\cmdline-tools
- android sdk            - D:\android-sdk
- jdk                    - E:\jdk-17.0.10
- proxy                  - http://127.0.0.1:7897
- gradle                 - E:\gradle-8.5

命令与脚本使用PowerShell格式。
需要联网下载环境相关，默认不使用代理，失败3次后才会使用代理。使用代理失败3次后，停下来等待用户操作。禁止使用各种workaround。
